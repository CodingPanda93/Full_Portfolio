What Are Databases?

A structured set of data

Database are a collection of information, they are used for creating, manipulating, and deleting data. There are many different types of databases and they can be classified in a basic approach of relational database or a non-relational database. Relations means a set of our data is related to anoter section of our data. Meanwhile in a non-relational database there is not a common related section of our data. Our databases are put into sections of data called shcemas, inside each schema each data set is organized into tables those tables contain columns and rows to house that piticular set of data.
